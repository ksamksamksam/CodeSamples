traceRay

This is the meat of our program. Takes a start position, a direction vector, and a depth number as input and outputs the color value at the final resting point of the ray. The method is only allowed to recurse maxDepth times, at which point it returns black as the output color.

Parameters:
startOrigin of our ray
rayDirection vector of our ray
